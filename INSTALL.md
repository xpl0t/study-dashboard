# Installationsanleitung

## Python installieren

Laden Sie sich eine Python-Version größer oder gleich *3.11* von der [Python Webseite](https://www.python.org/downloads/) herunter und installieren Sie diese.

## Laufzeitabhängigkeiten installieren

Die Anwendung erfordert die Laufzeitabhängigkeiten *textual* und *dataclasses-json*. Diese Abhängigkeiten werden mit folgendem Kommandozeilenbefehl installiert:

`pip install -r requirements.txt`

## Anwendung ausführen

Die Anwendung wird mit folgendem Befehl gestartet:

`python main.py`
